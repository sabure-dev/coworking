Работа электронного дневника:

важные моменты для родителей


Уважаемые родители, многие из вас сталкиваются с определёнными трудностями при использовании системы электронного дневника. Наш лицей обращается к вам с просьбой проявить немного терпения, поскольку любые задержки связаны исключительно с техническими аспектами функционирования системы. Убедительно просим проявлять понимание, ведь мы делаем всё возможное для оперативного устранения возникающих трудностей.

Подробные инструкции и разъяснения по работе с системой представлены в специальном разделе портала «Петербургское образование»: Портал «Петербургское образование» Раздел «Помощь» Ссылка: https://petersburgedu.ru/help

Также, если у вас устройство на платформе Android (версии ОС не ниже 10.0), воспользуйтесь прямой ссылкой для загрузки файла APK (без использования магазина приложений): Файл APK для установки приложения: https://service.petersburgedu.ru/api/file/id/894915e851c38a3ec8513286c992a464cb6a433e

Кроме того, подробную информацию о функционировании электронного дневника можно найти на нашем официальном сайте лицея в соответствующем разделе: Официальный сайт лицея  Родителям и ученикам  Электронный дневник Ссылка: https://www.xn- - 373- qddohl3g.xn- - p1ai/dnevnik.html

Обратите внимание , что воспользоваться возможностями электронного дневника можно не только через мобильное приложение, но и напрямую через веб- сайт портала «Петербургское образование».

Используя электронный дневник, вы получаете удобный инструмент для постоянного контроля успеваемости ваших детей и своевременного информирования обо всех важных событиях учебного процесса.

С уважением, администрация лицея!
